JavaScript Documentation

A simple guide to understanding JavaScript basics

What is JavaScript?

JavaScript is a dynamic, high-level programming language that is commonly used for creating interactive effects within web browsers. It is a core technology of the World Wide Web, alongside HTML and CSS.

JavaScript Basic Syntax

JavaScript code is written inside a <script> tag or in external JavaScript files. Here's an example:

    
    // This is a comment
    var message = "Hello, World!";
    alert(message); // Displays a message in a popup

In the above code, we declare a variable message and display it using the alert() method.

JavaScript Functions

Functions allow us to group code together and reuse it. Here's an example:

    
    function greet(name) {
        return "Hello, " + name + "!";
    }
    
    var greeting = greet("Alice");
    alert(greeting); // Displays "Hello, Alice!"

In this code, the greet() function takes an argument name and returns a greeting message.

JavaScript Loops

Loops allow us to execute code multiple times. Here's an example of a for loop:

    
    for (var i = 0; i < 5; i++) {
        console.log("Iteration: " + i);
    }

This loop will print out "Iteration: 0" to "Iteration: 4" in the console.
